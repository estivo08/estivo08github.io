Mac Auto-Typer Guide

Follow these steps to run a script that simulates human typing in any application.

Step 1: Copy the AppleScript Code

This is the code that will do the typing. You can change the text inside the quotes to whatever you want.

AppleScript Code 
delay 3

-- Paste your multi-line text between the quotes below:
set myText to "--> REPLACE THIS WITH YOUR TEXT <--"

-- This loop types out the text
tell application "System Events"
	repeat with c in characters of myText
		keystroke c
		-- This number controls the typing speed. 
		-- Bigger = slower. (e.g., 0.05)
		delay 0.01
	end repeat
end tell

Step 2: Run the Code in Script Editor

  1. Open the "Script Editor" app on your Mac (you can find it in Applications > Utilities).
  2. Paste the code you just copied into the blank white window.
  3. Click the Run button (it looks like a "Play" icon) in the toolbar.

Step 3: Give Permission (First Time Only)

The very first time you run this, your Mac will stop it and ask for permission to control your computer. This is normal!

  1. A pop-up will appear. Click "Open System Settings".
  2. It will take you to Privacy & Security > Accessibility.
  3. Find "Script Editor" in the list and turn the switch ON. You may need to enter your password.

Step 4: How to Use It

  1. Open any text field (like a Google Doc or a text file).
  2. Click your mouse cursor exactly where you want the typing to start.
  3. Switch back to the Script Editor app.
  4. Click the Run (play) button.
  5. It will wait 3 seconds and then start typing in your other window!
